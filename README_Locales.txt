KASWARE  v4.2.1 is delivered with support for 15 languages.

Full translations provided by kasware include:
English UK - the default language set.
English - USA
French - European

Not all locales are completely translated.

The locale files shipped with this version of KASWARE  are the 
latest at the date of this release.

Latest locales can be found here: http://kasware.com/

Application Locales (Default):
beans_messages.properties is the Java generic message file
data_messages.properties is relevant to the data interface 
pos_messages.properties is the MAIN application label and Messages file

Reports:
Each report has a .properties file associated with it.
You will find the files in the KASWARE  installation folder in the
kasware instalation sub-folder \reports\com\openbravo\reports

HOW TO CHANGE YOUR LANGUAGE
All languages are now compiled into the KASWARE  executable.
Any required locale file changes will require the project to be recompiled.

Set the KASWARE  Configuration>Locale to your required language
